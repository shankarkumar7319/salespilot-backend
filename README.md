🚀 SalesPilot – E-Commerce Management System

SalesPilot is a full-stack e-commerce web application that enables users to browse products, manage carts, and place orders with secure online payments. It also provides an admin dashboard to manage products, users, and overall business operations.

📌 Features
👤 Customer Features
      User Signup & Login (JWT Authentication)
      Browse products (Flipkart-style UI)
      Add to Cart & Manage Cart (update, delete, quantity control)
      Secure payment integration using Razorpay
      Order placement and tracking
      (Order → Shipping → Out for Delivery → Delivered)
      View order history and transaction details

👨‍💼 Admin Features
      Admin Login with role-based access
      Dashboard with:
      Total Users
      Total Orders
      Total Revenue
      Add / Update Products
      View all products
